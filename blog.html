Code Journey

As part of my blog i've decided to start this section where i'll be sharing my learning experience,
i'll also be documenting some of the things i learn.
Maybe even share some code snippets and hopefully some few months from now i can look back and see where it all began.

April 19th 2020,

It's been a month now since schooling got suspended as a result of the covid-19 pandemic, so i decided to use this opportunity to learn some skills. So far i've been able to get HTML and CSS certificates from sololearn, it's the little steps right ;) I also realised my deep interest in web development, talk about discovering yourself. *Silent laughs*. I am also working on building my personal portfolio, it's a bit overwhelming i must say, they so much to learn and do, so many example websites that challenge me, i can't count the number of times i have deleted my code feeling that this is not it, but i'm learning that the little things matter, and as they say Rome was not built in a day. So with this in mind i've decided to take what i have and keep making it better little by little. I also discovered bootsrap today, tbh i had no idea anything like this existed before. It's a front end framework for faster and easier web development, what's more it's mobile-first and it has built-in features for making web designs that allow websites to automatically adjust themselves to look good on devices of all sizes. WOOOOOOOW. Okay this looks interesting.. So the next couple of weeks i'll be learning javascript and bootsrap. Let's see how i can make my websites better with this.

Good reads

Deception Point

If you've read any of Dan Brown's books then i'm sure you know what an amazing writer he is.
This book is about deception as its title suggests, Rachael Sexton, an inteligence analyst is dispatched by the white house to verify the authenticity of NASA'S recent discovery. Rachael and Tolland, an expert accompanying her, discover evidence of scientific trickery. In their quest to discover the mastermind, they face attacks from assassins who will stop at nothing to ensure that the discovery is not sabotaged. I love the way this book keeps you guessing what will happen next with every page turn, the plot twists are top notch. If you are into mystery-thriller novels then you should definitely check this out. It's awesome.
